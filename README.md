# 상품 관리 서비스

## 요구사항 분석
- 상품 도메인 모델
    - 상품 ID (itemId)
    - 상품명 (itemName)
    - 가격 (itemPrice)
    - 수량 (itemQuantity)
- 상품 관리 기능
    - 상품 목록
    - 상품 상세
    - 상품 등록
    - 상품 수정
